Setup

```bash
# Kloniraj repo
git clone <repo-url>
cd eldar-muratovic-praksa

# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm run dev
```

## Portovi

- Backend: http://localhost:5000
- Frontend: http://localhost:3000 
- Promijeni u .env fajlovima ako je potrebno
